# Heart Disease Prediction System

## Objective

The objective of this project is to predict the presence of heart disease in a patient using machine learning techniques based on medical attributes.

## Dataset

Heart Disease Dataset (UCI)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Algorithm

Random Forest Classifier

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Handling Categorical Features
4. Label Encoding
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Prediction
9. Model Saving

## Features Used

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Rest ECG
* Maximum Heart Rate
* Exercise Induced Angina
* Oldpeak
* Slope
* Number of Major Vessels
* Thalassemia

## Results

* Accuracy Achieved: 98.54%
* Classification Report Generated
* Confusion Matrix Visualized
* Feature Importance Analysis Performed
